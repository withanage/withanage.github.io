10 * @class PKPComponentRouter
11 *
12 * @ingroup core
13 *
14 * @brief Class mapping an HTTP request to a component handler operation.
15 *
16 *  We are using an RPC style URL-to-endpoint mapping. Our approach follows
17 *  a simple "convention-over-configuration" paradigm. If necessary the
18 *  router can be subclassed to implement more complex URL-to-endpoint mappings.
19 *
20 *  For servers with path info enabled the component URL has the following elements:
21 *
22 *       .../index.php/context1/context2/$$$call$$$/path/to/handler-class/operation-name?arg1=...&arg2=...
23 *
24 *  where "$$$call$$$" is a non-mutable literal string and "path/to" is
25 *  by convention the directory path below the "controllers" folder leading to the
26 *  component. The next element ("handler-class" in this example) will be mapped to a
27 *  component class file by "camelizing" the string to "HandlerClassHandler" and adding
28 *  ".php" to the end. The "operation-name" is transformed to "operationName"
29 *  and represents the name of the handler method to be called. Finally "arg1", "arg2",
30 *  etc. are parameters to be passed along to the handler method.
31 *
32 *  For servers with path info disabled the component URL looks like this:
33 *
34 *       .../index.php?component=path.to.handler-class&op=operation-name&arg1=...&arg2=...
35 *
36 *  The router will sanitize the request URL to a certain amount to make sure that
37 *  random code inclusions are prevented. User authorization and parameter validation
38 *  are however not the router's concern. These must be implemented on handler level.
39 *
40 *  NB: Component and operation names may only contain a-z, 0-9 and hyphens. Numbers
41 *  are not allowed at the beginning of a name or after a hyphen.
42 *
43 *  NB: Component handlers must implement an initialize() method that will be called
44 *  before the request is routed. The initialization method must enforce authorization
45 *  and request validation.
46 */

393    public function _retrieveServiceEndpointParts(PKPRequest $request)
